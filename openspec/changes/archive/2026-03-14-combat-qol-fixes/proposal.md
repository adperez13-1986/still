# Combat QoL Fixes

Bundle of quality-of-life improvements and bug fixes discovered during playtesting.

## Changes

1. **Effective damage display** — Enemy intent shows actual damage after scaling, Strength, and Weak modifiers instead of base value
2. **Pile viewer** — Tap Draw/Discard/Exhaust counters during combat to inspect pile contents
3. **Pile counters on mobile** — Draw/Discard/Exhaust counts now visible on mobile (were desktop-only)
4. **Shop info button** — INFO button in shop opens deck/equipment overlay so player can see what they have before buying
5. **Carry-nothing bug fix** — "Carry nothing" now properly clears the carried part instead of silently keeping it
