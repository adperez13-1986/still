## REMOVED Requirements

### Requirement: Findable action pool
**Reason:** The concept of "findable actions" is replaced by card/part/equipment drops from the existing deckbuilder drop system. The ActionDefinition type and its pool are deleted.

### Requirement: Action replacement
**Reason:** No action slots to replace actions into. Card additions to the deck work via the existing deckbuilder pattern (cards accumulate; can be removed at rest sites or shops).
