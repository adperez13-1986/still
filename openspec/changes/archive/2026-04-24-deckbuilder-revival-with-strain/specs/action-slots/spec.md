## REMOVED Requirements

### Requirement: Unified action slot system
**Reason:** The 5-slot paired system is replaced by the body-slot deckbuilder revival. Slot arrangement, pair links, and per-slot push toggles are no longer part of the combat system. The concept "action slots" does not exist in the new system.

### Requirement: Push cost per slot
**Reason:** Replaced by per-card push model (see `card-system` spec). Pushing is per-card, not per-slot.

### Requirement: Slot rearrangement between fights
**Reason:** No action slots to rearrange. Build identity comes from deck/equipment/parts composition.
